Multi-robot autonomous exploration in 2D

event 2023-06-10 (Day 2) ~ 14:30 - 15:00
explore Grand Hall A
  • comment English
  • network_check Intermediate

This project demonstrates that multiple robots can collaborate to explore an unknown environment without GPS to generate a map with reasonable accuracy. This addresses exploration as a problem in the planning layer of the robotics domain. A frontier-basedÂ  exploration methodology will be explained. The main approach uses Rapidly-expanding Random Trees for frontier detection, and map merging for information sharing. The developed algorithm also makes use of existing solutions for SLAM, and path planning. The algorithm runs in a network of robots, and a master machine on ROS. The project shows that using multiple robots can reduce exploration time compared to using a single robot.
